# Phase 02.1 Plan 03: scoped GSAP scroll-telling for the OT-2 deep-dive

**Vendored ScrollTrigger + a gsap.context lifecycle scoped to the overlay scroller that draws the gel motif, draws the deck plan, assembles the CAD parts, and reveals the RUN/OUTCOME beats on scroll — fully suppressed under reduced motion, cleanly torn down on close.**

## Accomplishments
- Vendored GSAP ScrollTrigger 3.15.0 locally (version-matched to the already-vendored core; one classic `<script>` after gsap core, before the module) — no CDN hot-link.
- `initOt2Motion()` / `killOt2Motion()`: `gsap.context(fn, scroller)` where `scroller = [data-proj="ot2"]`; built at the `applyTrans` `t>=1` completion for `into.id==='ot2'` (after transform/filter cleared) then `ScrollTrigger.refresh()`; reverted in `closeProj`.
- Choreography (transform/opacity/stroke-dashoffset only): masthead lanes/bands draw-on + subtle pipette load loop; deck-plan draw-on (scrub); CAD parts stagger-assemble (scrub); RUN figure lift + OUTCOME numbers rise (scrub).
- Reduced-motion: single `REDUCED` gate — no timeline builds, every section fully readable; `+ @media (prefers-reduced-motion: reduce)` CSS block.
