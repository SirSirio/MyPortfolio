## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 - Blocking] CAD blueprint wrapper switched off `.media-slot`**
- **Found during:** Task 2 (deep-dive BUILD section)
- **Issue:** The plan suggested reusing the `.blueprint` CAD plate; wrapping it in `.media-slot` (as the home composite does) would have left the `.media-slot:not(:has(> img, > video, ...))` placeholder striping behind the renders, because the `<img>`s are grandchildren inside `.blueprint`, not direct children.
- **Fix:** Put the blueprint in a dedicated `.dd-figure` wrapper (no `.media-slot` class) and gave it an `aspect-ratio: 2/1` so the 2├Ś2 render grid sizes correctly outside the composite flex row.
- **Files modified:** site/index.html, site/styles.css
- **Verification:** Task 2 grep gate PASS; renders reference the four existing `ot2-part-*.png` first-party assets.
- **Committed in:** `a95f56b` (Task 2 commit)

- **Found during:** Task 2 verification
- **Issue:** The overlay accent rule was authored as `[data-proj='ot2']` (single quotes); the plan's verify gate and artifact contract require the literal `[data-proj="ot2"]` (double quotes).
- **Fix:** Changed the selector to double quotes.
- **Files modified:** site/styles.css
