<objective>
Build the end-to-end warp path to a real OT-2 deep-dive. After this plan, a visitor
can click the "Automated Gel Electrophoresis" card on the home page, ride a deliberate
warp/hyperlight transition, land on a full, readable DALSA OT-2 case study (the problem,
the design approach, the build, the run, the outcome), scroll it, and return home through
an equally intentional transition. This is the NAV-03 deliverable and the vertical slice
that Phases 4 reuses in shape.

The warp machinery already exists — a complete, reduced-motion-aware, palette-aware
transition system in the V4 design snapshot (`Sirio V4 - Deep Field x Hyperlight.dc.html:266-520`).
This plan PORTS it into the live `site/main.js` (DC class → ES module), adds the
`curProj` engine-pause guard the site loop currently lacks (D-20), wires the OT-2 card
(D-04), builds the `[data-proj="ot2"]` overlay, and fills it with real curated content
from the DALSA report + profile.md (D-05..D-10). No code template — a hand-crafted pilot (D-01).

Purpose: give NAV-03 the deeper page it was stranded without, and prove the case-study
shape with one real page.
Output: a working home↔deep-dive warp and a real, media-carrying OT-2 case study.
</objective>
