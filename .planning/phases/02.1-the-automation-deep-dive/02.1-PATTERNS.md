### `site/main.js` — warp transition port (engine, event-driven + rAF)

**Analog:** `Sirio V4 - Deep Field x Hyperlight.dc.html:394-484` (the `openProj`/`closeProj`/`applyTrans` methods) — port from DC class methods (`this.trans`, `this.curProj`, `this.eng.*`) into module-scope functions/state, resolving `this.eng.X` → the existing named imports `clamp`, `tintAt`, `rgba` from `./star-engine.js` (already imported at `site/main.js:11`, currently `{ HeroStar, REDUCED, DPR, tintAt, rgba }` — add `clamp`).

**openProj — reduced-motion branch + trans setup** (`.dc.html:394-410`):
```javascript
openProj(id) {
  if (this.trans) return;
  const target = this.projEl(id); if (!target) return;
  const fromId = this.curProj && this.curProj !== id ? this.curProj : null;
  const pos = parseFloat(target.dataset.pos);        /* destination palette position */
  if (this.reduced) {                                 /* PORT: gate on REDUCED import */
    target.style.visibility = 'visible'; target.style.opacity = '1'; target.scrollTop = 0;
    document.body.style.overflow = 'hidden'; this.curProj = id; return;
  }
  this.trans = { start: performance.now(), dur: this.jumpMs(), pos: isNaN(pos)?0:pos,
                 max: 0.016 * this.warpMax(), out: fromId?{id:fromId}:{isField:true}, into:{id} };
  this.curProj = id;
}
```

**applyTrans — the masked MID-POINT DOM swap** (`.dc.html:455-464`) — the load-bearing bit:
```javascript
/* mid-point swap (masked by the warp field) — happens at t>=0.5, hidden by drawWarp */
if (t >= 0.5 && !tr._mid) {
  tr._mid = true;
  if (!tr.out.isField && outEl) {
    outEl.style.visibility = 'hidden'; outEl.style.opacity = '0';
    const oi = outEl.querySelector('[data-proj-inner]'); if (oi){ oi.style.transform=''; oi.style.filter=''; }
  }
  if (!tr.into.isField && inEl){ inEl.style.visibility='visible'; inEl.scrollTop=0; document.body.style.overflow='hidden'; }
  if (tr.into.isField) document.body.style.overflow = '';
}
```
Completion branch (`.dc.html:475-483`): when `t >= 1`, clear the inner transform/filter and set `this.trans = null`. **This is the hook point for `initOt2Motion()`** (RESEARCH Pattern 3): call it here when `tr.into.id === 'ot2'`, AFTER the transform/filter are cleared, then `ScrollTrigger.refresh()`.

**drawWarp — palette-aware tint** (`.dc.html:487-520`): paints the `[data-warpc]` overlay canvas only while `this.trans || this.burst` is active; idle branch clears it and sets `warpov.opacity='0'`. Tint = `this.eng.tintAt(pos)` where `pos = act.live ? this.scrollPos() : act.pos`. `data-pos` on the overlay (RESEARCH §Code Examples, low value ≈ blue/gold end) drives this.

```javascript
function loop(now) {
```

This is the D-20 invariant: the deep-field engine pauses so GSAP owns the frame budget. Reconcile it deliberately (Pitfall 2).

### `site/index.html` — OT-2 card becomes clickable (markup / trigger)

**Analog:** `.dc.html:307-314` (open/close wiring) + the existing card structure.

The OT-2 plate is `<article data-reveal="1" class="plate plate--flip">` at **`site/index.html:237`** (the "Portable Precision Liquid Dispenser" plate at `:200` is a different card — do NOT wire that one; only OT-2 per D-04). Add `data-open="ot2"` and A11y affordances.

**Wire ONCE at boot** (RESEARCH §Code Examples, adapted from the per-frame `acquire()` at `.dc.html:307-314` — the site has no per-frame `acquire`, so wire once in `boot()`):
```javascript
document.querySelectorAll('[data-open]').forEach(b => {
  b.addEventListener('click', (e) => { e.preventDefault(); openProj(b.dataset.open); });
  b.addEventListener('keydown', (e) => {                      // A11y — Pitfall 3
    if (e.key === 'Enter' || e.key === ' ') { e.preventDefault(); openProj(b.dataset.open); }
  });
});
document.querySelectorAll('[data-close]').forEach(b =>
  b.addEventListener('click', (e) => { e.preventDefault(); closeProj(); }));
```
The DC design uses a `__wired` guard because `acquire()` re-runs every frame; the site wires once at boot, so the guard is unnecessary. Give the card `role="button"` + `tabindex="0"` + an accessible label (Pitfall 3 — do not ship a keyboard trap).

## Shared Patterns

### Reduced-motion — ONE source of truth
**Source:** `site/star-engine.js:12` → `export const REDUCED = window.matchMedia('(prefers-reduced-motion: reduce)').matches;`
**Apply to:** the warp (`openProj`/`closeProj` reduced branches, `.dc.html:399-403/415-417`) AND every GSAP timeline.
**Rule (RESEARCH Pattern 4 + `main.js:139-146`):** gate on the imported `REDUCED`. Do NOT add `gsap.matchMedia()` or a second `matchMedia('(prefers-reduced-motion…)')` — 02-CONTEXT D-36. The site's reveal contract (`main.js:141-160`) is the template: under `REDUCED`, present everything and never build motion; JS is the ONLY thing that hides, so a JS failure or reduced-motion visitor sees the whole page.

### No layout reads in the loop/pointer handler
**Source:** `site/main.js:28-34` (cached `heroRect`), `:177-184` (`onPointerMove` stores target only).
**Apply to:** GSAP measurement stays inside its own ScrollTrigger/`ScrollTrigger.refresh()`, never in the engine's `loop()`/`applyPointer()`/`onScroll()`.

### Media contract — first-party, committed, relative paths, always poster a video
**Source:** `site/index.html:249-252` (the shipped OT-2 timelapse `<video>` with `poster`, `muted loop playsinline`, no `autoplay`) + `main.js:245-248` (playback started from JS only when motion allowed).
**Apply to:** the deep-dive's timelapse + the two new PNGs. Never a remote `src` — this same rule is why GSAP is VENDORED, not hot-linked (RESEARCH §Standard Stack).
